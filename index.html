Dokumentasi Sistem

Mekanisme Otentikasi
Terdesentralisasi.

Sistem ini tidak menggunakan username atau password tradisional yang rentan diretas. Kami menggunakan Kriptografi Wallet sebagai identitas digital.

Prinsip #1

Anonimitas

Kami hanya membaca Public Address Anda. Tidak ada data pribadi (Email/No HP) yang disimpan.

Prinsip #2

Keamanan

Hanya wallet yang terdaftar (Whitelisted) yang dapat mendekripsi konten latihan soal.

1 Persiapan Perangkat

Sebelum mengakses, Anda memerlukan "Dompet Digital" sebagai kunci. Pilih salah satu ekosistem:

MetaMask (EVM)

Standard global. Mendukung Ethereum, BSC, Polygon, dll. Tersedia di iOS/Android & Chrome.
Phantom (Solana)

Sangat cepat dan ringan. Pilihan terbaik jika Anda menyukai ekosistem Solana.

Tips Keamanan: Gunakan "Burner Wallet" (Wallet baru/kosong) jika Anda hanya ingin mengakses materi ini. Jangan gunakan wallet utama yang berisi aset berharga untuk browsing sembarangan.

2 Proses Whitelist

Agar sistem mengenali Anda, alamat wallet (Public Key) Anda harus didaftarkan ke database kami.

  1. Buka aplikasi Wallet Anda.
  2. Salin alamat Anda (Contoh: 0x71C...9A2 atau G5j...k9L).
  3. Kirim alamat tersebut ke Admin melalui DM Instagram atau WhatsApp.

3 Metode Akses

METODE A

Auto-Connect (Desktop)

Jika Anda menggunakan Laptop/PC dengan ekstensi Chrome terpasang.

  • 1. Buka halaman utama.
  • 2. Klik tombol "EVM / METAMASK" atau "SOL / PHANTOM".
  • 3. Pop-up ekstensi akan muncul.
  • 4. Klik "Sign" atau "Connect" untuk memverifikasi kepemilikan.
METODE B

In-App Browser (Mobile)

Browser biasa (Chrome/Safari) di HP tidak bisa mendeteksi aplikasi wallet secara langsung.

  • 1. Buka aplikasi MetaMask atau Phantom di HP.
  • 2. Cari menu "Browser" (ikon bola dunia) di dalam aplikasi.
  • 3. Masukkan URL web kami di sana.
  • 4. Login seperti biasa.
METODE C

Manual Input (Universal)

Gunakan ini jika Anda di warnet, perpustakaan, atau meminjam perangkat teman.

  • 1. Salin alamat wallet Anda dari catatan/HP.
  • 2. Pada halaman login, lihat bagian "Atau Manual".
  • 3. Tempel alamat Anda di kolom input.
  • 4. Klik "VERIFIKASI".
